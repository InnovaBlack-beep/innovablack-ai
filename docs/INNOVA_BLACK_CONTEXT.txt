SERVICIO 01 — DTX Launch™
  Para: Constitución de nueva SOFOM, SOFIPO o IFPE
  Qué incluye:
    - Arquitectura tecnológica completa
    - Flujos operativos diseñados
    - Manuales CNBV (onboarding, seguridad, PLD, continuidad de negocio)
    - Setup PLD/KYC desde cero
    - Conexión a infraestructura regulatoria
    - Documentación lista para presentar ante CNBV
  Duración: 3–6 semanas

SERVICIO 02 — DTX Compliance Engine™
  Para: Automatización de compliance en institución existente
  Qué incluye:
    - Automatización IA de reportes ROU/ROP a la UIF
    - Monitoreo transaccional inteligente
    - Reducción de falsos positivos en alertas PLD
    - Onboarding digital KYC (minutos, no días)
    - Integración CNBV, UIF, SAT
  Resultado prometido: -70% en tiempo de compliance
  Duración: 2–5 meses

SERVICIO 03 — DTX Scale™
  Para: Institución que quiere escalar al siguiente nivel regulatorio
  Qué incluye:
    - Conexión a SPEI como participante directo o indirecto
    - BIN sponsor para emisión de tarjetas
    - Onboarding digital de clientes a escala
    - Scoring crediticio con IA
    - Arquitectura multi-producto (SOFOM + SOFIPO, IFPE + banco)
  Duración: 4–8 meses

SERVICIO 04 — DTX Upgrade™
  Para: Institución existente con sistemas legacy o procesos manuales
  Qué incluye:
    - Migración de sistemas legacy a plataformas modernas
    - Rediseño de procesos manuales
    - Preparación para Ley Fintech 2.0
    - Preparación para evaluación GAFI
    - Implementación de infraestructura normativa requerida
  Duración: 2–6 meses

SERVICIO 05 — DTX Retainer™  [EL MÁS DIFERENCIADOR]
  Para: Cualquier institución que quiera un equipo tech-estratégico externo
  Descripción: El único servicio de CTO/COO externo para instituciones
               financieras reguladas en México. No existe en el mercado.
  Qué incluye:
    - Monitoreo regulatorio en tiempo real (CNBV, UIF, Banxico, SAT)
    - Actualización ante nuevas disposiciones y circulares
    - Formación continua del equipo (compliance, operaciones, directivos)
    - Advisory mensual directo al CEO
    - Gestión de proveedores tecnológicos
    - Respuesta ante visitas o requerimientos de autoridades

PROCESO DE 5 FASES (para todos los servicios):
  01 Diagnóstico      → Evaluación + reporte ejecutivo + roadmap 90 días
  02 Diseño Framework → Arquitectura tech + flujos + estructura regulatoria
  03 Implementación   → Sistemas + automatización + integraciones + documentación CNBV

  - Diagnóstico gratuito de 45 minutos

TECNOLOGÍA:
  - HTML5 / CSS3 / JavaScript vanilla (sin frameworks — sitio estático)
  - Fuentes desde Google Fonts: Libre Baskerville + Instrument Sans + DM Mono
  - Sin dependencias externas adicionales (sin jQuery, sin React)

ARQUITECTURA DE ARCHIVOS:
  - Un solo archivo HTML por ahora (todo inline — CSS en <style>, JS en <script>)

  - Documentos descargables: /assets/docs/

  - Variables CSS definidas en :root (ya establecidas, no cambiar nombres)
  - Clases BEM-like: .sec-nombre, .card-elemento, .btn-variante
  - Scroll reveal: clase .r en elemento, agregar .in con IntersectionObserver
  - Delays de animación: .d1 (.08s), .d2 (.16s), .d3 (.24s), .d4 (.32s)

  3. Agregar clase .on-dark si la sección es sobre fondo negro
  4. Todos los elementos animados llevan clase .r (reveal)
